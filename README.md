## Descripción 📄
Este portafolio fue desarrollado por AdrielGBM utilizando HTML, SCSS y TypeScript para crear una interfaz moderna, funcional y escalable. Se empleó Node.js para gestionar dependencias de desarrollo, integrando herramientas como Gulp para automatizar tareas y optimizar recursos, así como ESLint para mantener un código limpio y estandarizado. Este proyecto refleja un enfoque en la organización, las buenas prácticas y la implementación de tecnologías actuales en el desarrollo web.
## ¿Qué he aprendido en este proyecto? ✍🏻
En este proyecto he trabajado en mejorar la organización y estructura del código aplicando la metodología BEM (Block, Element, Modifier), lo que me ha permitido mantener un diseño escalable y fácil de mantener. Además, he aprendido a utilizar tecnologías como SCSS y TypeScript, que han sido fundamentales para escribir estilos y lógica más eficientes, modulares y robustos en comparación con CSS y JavaScript puros.

También he profundizado en el uso de Gulp como herramienta de automatización, integrándolo con Node.js para compilar y optimizar los recursos del proyecto. Esto no solo ha mejorado mi flujo de trabajo, sino que también ha reforzado mis habilidades para gestionar tareas complejas en proyectos más grandes. En conjunto, esta experiencia me ha permitido consolidar prácticas modernas en el Desarrollo Front-End.
